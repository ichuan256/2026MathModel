# 2023B题文件使用说明

本文档面向第一次接触本题的人，说明各文件夹中脚本的用途、运行顺序和输出结果。所有与 2023B 题相关的文件都放在 `2023B题` 文件夹内，四个子文件夹分别对应问题一到问题四。

## 一、推荐运行顺序

建议按题目顺序运行：

```powershell
python D:\CaoTX\2026数模\2023B题\问题一\solve_2023b_q1.py
python D:\CaoTX\2026数模\2023B题\问题二\solve_2023b_q2.py
python D:\CaoTX\2026数模\2023B题\问题三\solve_2023b_q3_angle_search.py
python D:\CaoTX\2026数模\2023B题\问题三\solve_2023b_q3_pso.py
python D:\CaoTX\2026数模\2023B题\问题四\make_contour_from_attachment.py
python D:\CaoTX\2026数模\2023B题\问题四\kmeans_depth_classification.py
python D:\CaoTX\2026数模\2023B题\问题四\solve_2023b_q4_kmeans_curved_lines.py
```

其中，问题四的三个脚本建议按上面顺序运行：先生成等深线图，再生成 K-means 分区图，最后生成曲线测线布设图。

## 二、文件夹结构

```text
2023B题
├─ 问题一
├─ 问题二
├─ 问题三
└─ 问题四
```

每个问题文件夹内保留该问题对应的方法脚本、结果表格和图像文件。一般来说，`.py` 是可运行脚本，`.xlsx` 和 `.csv` 是数据结果，`.png` 和 `.svg` 是图片结果。

## 三、问题一

### 脚本

`问题一/solve_2023b_q1.py`

用途：计算不同测线位置处的海水深度、覆盖宽度等指标，并输出问题一结果表和图像。

### 主要输出

`问题一/result1.xlsx`

问题一表格结果。

`问题一/q1_multibeam_profile.png`

问题一可视化图片，适合直接查看或放入文档。

`问题一/q1_multibeam_profile.svg`

问题一矢量图，适合论文或报告中放大使用。

## 四、问题二

### 脚本

`问题二/solve_2023b_q2.py`

用途：根据问题二建模结果，计算不同测线方向角和位置下的覆盖宽度，并绘制覆盖宽度热力图。

### 参考文件

`问题二/multibeam_q1_corrected_q2_model.pdf`

问题二建模说明文件。

`问题二/multibeam_q2_text.txt`

### 主要输出

`问题二/result2.xlsx`

题目格式对应的结果表。

`问题二/q2_coverage_result.xlsx`

覆盖宽度计算结果表。

`问题二/q2_coverage_heatmap.png`

覆盖宽度热力图。

`问题二/q2_coverage_heatmap.svg`

覆盖宽度矢量热力图。

## 五、问题三

问题三目前保留两种方法：一维角度搜索法和改进粒子群算法。两种方法互相独立，可以分别运行和比较。

### 方法一：角度搜索

脚本：

`问题三/solve_2023b_q3_angle_search.py`

用途：对测线方向角进行搜索，并用局部精修得到较优测线布设方案。

主要输出：

`问题三/q3_angle_search_layout.png`

角度搜索法测线布设图。

`问题三/q3_angle_search_layout.svg`

角度搜索法矢量测线布设图。

`问题三/q3_angle_search_lines.csv`

角度搜索法最终测线数据。

`问题三/q3_angle_search_history.csv`

搜索过程记录。

### 方法二：改进粒子群算法

脚本：

`问题三/solve_2023b_q3_pso.py`

用途：使用改进粒子群算法优化测线方向角、测线位置和覆盖约束，输出最终布设图。

主要输出：

`问题三/q3_pso_layout.png`

改进粒子群算法测线布设图。

`问题三/q3_pso_layout.svg`

改进粒子群算法矢量测线布设图。

`问题三/q3_pso_lines.csv`

改进粒子群算法最终测线数据。

### 使用建议

如果只是需要一个稳定、可解释的结果，先看 `q3_angle_search_layout`。如果需要体现智能优化算法，则重点看 `q3_pso_layout` 和 `q3_pso_lines.csv`。

## 六、问题四

问题四从附件中的离散测深点开始，先生成连续等深线图，再用三维 K-means 进行区域划分，最后在分区基础上进行曲线测线布设。

### 输入文件

`问题四/附件.xlsx`

离散测深点数据，是问题四最重要的输入。

`问题四/B题.pdf`

原题文件。

`问题四/B题_text.txt`

原题文字内容整理。

### 第一步：生成等深线图

脚本：

`问题四/make_contour_from_attachment.py`

用途：读取 `附件.xlsx`，识别东西方向坐标、南北方向坐标和测深矩阵，生成离散点表和等深线图。

主要输出：

`问题四/attachment_bathymetry_points.csv`

从附件中整理出的离散测深点表。

`问题四/attachment_contour_map.png`

等深线图。

`问题四/attachment_contour_map.svg`

等深线矢量图。

### 第二步：三维 K-means 分区

脚本：

`问题四/kmeans_depth_classification.py`

用途：使用 `x、y、水深` 三维特征进行 K-means 聚类，并用肘部法选择聚类数量。这里不是只按水深聚类，而是同时考虑点的空间位置和深度。

主要输出：

`问题四/attachment_kmeans_elbow.csv`

不同 K 值的 SSE 记录和最终选取的 K。

`问题四/attachment_kmeans_depth_clusters.csv`

每个测深点所属的 K-means 分区。

`问题四/attachment_kmeans_depth_zones.png`

三维 K-means 分区图。

`问题四/attachment_kmeans_depth_zones.svg`

三维 K-means 分区矢量图。

### 第三步：曲线测线布设

脚本：

`问题四/solve_2023b_q4_kmeans_curved_lines.py`

用途：在三维 K-means 分区基础上，用改进粒子群算法优化每个区域的测线重叠参数，并沿等深线生成曲线测线。测线长度、覆盖区域和重叠区域均按空间分区判断。

主要输出：

`问题四/q4_kmeans_curved_lines.csv`

最终曲线测线数据。

`问题四/q4_kmeans_curved_summary.csv`

问题四总长度、测线数量、重叠惩罚和迭代信息。

`问题四/q4_kmeans_curved_lines.png`

问题四曲线测线布设图。

`问题四/q4_kmeans_curved_lines.svg`

问题四曲线测线布设矢量图。

## 七、图片格式说明

每个主要图像一般同时输出 PNG 和 SVG。

PNG：适合快速预览、插入普通文档。

SVG：矢量图，适合放大查看、论文排版或后期编辑。

如果发现 PNG 放大后模糊，应优先使用同名 SVG。

## 八、运行环境

建议使用带有常用科学计算库的 Python 环境，例如 Anaconda。脚本主要依赖：

```text
numpy
openpyxl
matplotlib
Pillow
```

如果运行时报缺少库，可以在当前 Python 环境中安装对应包。

## 九、常见注意事项

1. 问题四必须保证 `附件.xlsx` 和三个问题四脚本在同一个文件夹内。
2. 问题四的最后一个脚本包含粒子群迭代，运行时间会明显长于前几个脚本。
3. 如果只想快速查看地形和分区，可以只运行问题四前两个脚本。
4. 如果修改了 `附件.xlsx`，需要重新运行问题四三个脚本，后面的分区和测线图才会同步更新。
5. 问题三和问题四都有优化算法，运行时终端会输出迭代进度，用于判断程序是否仍在正常运行。
