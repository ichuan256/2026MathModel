# 第五问优秀论文附录代码复现

## 1. 复现对象

本次复现对象不是根据论文正文重新设计的模型，而是网页附录第 89～96 页给出的 MATLAB 文件 `fifth.m`。

Python 代码逐项保留以下实现细节：

1. 真目标仅离散上下底面圆周，共 600 个点；
2. 每架无人机根据高度相同时的水平距离选择最近导弹；
3. 每架无人机只优化一组航向、速度、第一枚投放时刻和统一引信延迟；
4. 三枚弹的投放时刻固定为第一枚时刻、第一枚时刻加 1 秒、第一枚时刻加 2 秒；
5. 每个变量分别取 20 个网格点，单架无人机枚举 20^4＝160000 个候选；
6. 最终指标为 15 枚烟幕弹各自有效时长之和。

## 2. 复现结果

Python 复现结果：

```text
21.077025532722473 s
```

21.0770 s

0.000025532722 s

差异完全来自正文保留四位小数，因此附录代码已经成功复现。

## 3. 与论文正文目标的核对

按照附录代码输出的单弹区间，分别对每枚导弹取时间并集：

```text
M1: [4.025410, 8.893192] 与 [20.316405, 24.056730]
M2: [16.987149, 20.696312] 与 [23.711475, 27.026691]
M3: [33.539913, 35.964816]
```

三个导弹区间没有公共交集，因此：

```text
三导弹共同遮蔽时长 = 0 s
```

所以 21.0770 s 是“15 枚弹单独时长之和”，不是论文正文公式所写的三导弹共同遮蔽时间交集。

## 4. 附录代码中的其他不一致

1. `fifth.m` 把 FY5 初始位置写成 `(13000, 2000, 1300)`，题面为 `(13000, -2000, 1300)`。
2. 优化阶段使用三枚弹相隔 1 秒的投放时刻，但写表阶段把前两枚弹的有效时长误当成后续投放时间偏移。
3. 最近导弹分配与正文所述整数规划并不一致，附录实际没有求解正文公式中的整数规划。
4. 附录将单弹时长直接累加，没有消除同一导弹内部的重叠，也没有计算三导弹交集。

## 5. 结论

优秀论文的数值 21.0770 s 已成功复现，但复现证明该数值来自附录代码的求和指标。它可作为“论文代码复现结果”，不能作为题意下三枚导弹共同有效遮蔽时长。

严格按第五问题意的当前可行结果仍应引用修正模型的 3.743312451644 s，而不能引用 21.0770 s。
